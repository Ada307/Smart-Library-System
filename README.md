## Smart Library Management System (Java Swing)
A desktop-based Library Management System built using **Java Swing** and **JDBC**.
The application includes separate dashboards for **Admin, Librarian and Students**, along with small pop-up windows (using JOptionPane) for alerts and messages.

## Features
Login screen for all users
Admin, Librarian and Student dashboards
Add / issue / return books
Manage users and records
Swing-based GUI with pop-up dialogs
MySQL database connectivity through **JDBC**

## Technologies Used
Java (Swing GUI)
JDBC + MySQL
OOP concepts
